## Installation guide

### 1. Clone repository

```bash
git clone https://github.com/antancelin/SongsExplorerDeezer-Frontend.git
```

```bash
cd SongsExplorerDeezer-Frontend
```

### 2. Frontend configuration

```bash
yarn
```

### 3. Start local website

```bash
yarn dev
```
